# PRD: Účtovníctvo Jedináková - One-Page Website

## Original Problem Statement
Create a one-page accounting website for "Účtovníctvo Ing. Zuzana Jedináková" based on provided PDF documents with Slovak language, green color scheme, stock photos.

## Architecture & Tech Stack
- **Frontend**: React.js with Tailwind CSS, Framer Motion animations, react-scroll for smooth navigation
- **Backend**: FastAPI (Python) with MongoDB
- **Design**: Green emerald color scheme (#047857), glassmorphism header, Outfit + Plus Jakarta Sans fonts

## User Personas
1. **Small Business Owners (SZČO)** - Need simple accounting services
2. **Company Representatives (s.r.o.)** - Need double-entry bookkeeping
3. **New Entrepreneurs** - Need business plan preparation, trade license assistance

## Core Requirements (Static)
- One-page responsive website in Slovak
- Sticky header with navigation and CTAs
- Contact form with backend integration
- Pricing tables with JÚ/PÚ toggle
- Mobile-friendly with sticky CTA buttons
